DASHBOARD >> PLUGIN >> iNSTALLED PLUGINS

WordPress Plugins

On Clicking Installed Plugins option, Your WordPress theme dashboard screen will look something like this:

On this screen you can view all installed plugins , activated/deactivated plugins and edit option for configuring the plugin.

Delete Plugin

In order to delete a plugin first you need to Deactivate , then you can Delete .

The other option is performing a bulk operation by choosing "Bulk Actions" .

While deactivating a plugin is not a bad practice, uninstalling is important if you don’t plan on using them again. This is due to plugin that you are not using on your site can be a potential liability.
